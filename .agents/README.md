# Agent Notes

This folder is for maintainer-facing notes and reusable agent workflows that should be visible in the public repository but should not dominate the human README.

Primary coding-agent instructions live in [../AGENTS.md](../AGENTS.md). Keep that file as the source of truth for setup, architecture, safety, tests, and release commands.

Rules for adding files here:

- Keep notes evidence-based and current.
- Do not include secrets, local paths, private logs, or unpublished credentials.
- Do not add instructions that bypass `AGENTS.md`, approval gates, release checks, or security policy.
- Prefer short runbooks over long internal transcripts.
