# Creative Design

## Purpose

Use this parent skill when a task asks for visual design, diagrams, branded assets, infographics, creative writing, humanized prose, themes, or canvas-based planning.

## Hierarchy Reading Rules

1. Identify whether the output is visual, textual, diagrammatic, brand-governed, or a mixed creative artifact.
2. Read only the child skill whose constraints affect the output format or style.
3. Keep source evidence and brand requirements separate from creative suggestions.
4. Verify created artifacts through the relevant render, preview, or inspection tool when available.

## Child Skill Guide

- Use architecture or Excalidraw diagrams for system, flow, and sketch-style diagrams.
- Use canvas design for node/edge planning and visual workspaces.
- Use brand guidelines, theme factory, and infographic design for polished visual systems and explainers.
- Use creative writing and humanized writing for prose, voice, style, and audience-sensitive drafts.

## Verification

- For visual artifacts, verify structure, labels, hierarchy, and source references.
- For writing, preserve requested tone and factual constraints.
