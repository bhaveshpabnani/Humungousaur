# Daily Planning Reference

## Evidence Order

1. User-provided constraints and current date/time context.
2. Explicit commitments and triggers.
3. Active goals, tasks, focus, and persona.
4. Recent memory and activity.
5. Environment constraints such as disk, provider readiness, or unavailable tools.

## Planning Output Shape

Use a compact structure:

- Now: one primary action.
- Next: two or three actions that follow naturally.
- Later: deferred work.
- Waiting: blocked items and unblock condition.
- Reminders: wakeups or follow-ups that should be recorded only with user intent.
