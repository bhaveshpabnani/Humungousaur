# Support

Humungousaur is an early alpha local-first desktop agent runtime. Please keep support requests evidence-based and avoid posting secrets, private logs, private screenshots, or account tokens.

## Where To Ask

- Bugs: open a GitHub issue with the bug template.
- Setup help: open a GitHub issue with the setup-help template.
- Security issues: follow [SECURITY.md](SECURITY.md) and report privately.
- Contributions: read [CONTRIBUTING.md](CONTRIBUTING.md) before opening a PR.

## Good Support Reports Include

- operating system and version
- install method
- Python version
- exact command or desktop action
- model provider and model, with secrets redacted
- expected behavior
- actual behavior
- relevant logs or screenshots with private data removed
- what you already tried

## Please Do Not Include

- API keys, tokens, private keys, cookies, or passwords
- private user data from channels, browsers, files, screenshots, or voice transcripts
- destructive reproduction steps against someone else's machine or account
- broad feature bundles without a concrete use case
