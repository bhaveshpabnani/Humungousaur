# Humungousaur README Motion Design

## Style Prompt

Create a calm but powerful product-architecture motion graphic for a local-first desktop agent. The look should feel like a modern macOS control surface meeting a trust-and-safety systems diagram: bright canvas, crisp typography, warm red identity, blue planning energy, green completion/memory signal, and visible approval gates. Motion should be precise, confident, and readable rather than flashy.

## Colors

- `#FAFAFA` canvas background
- `#111827` primary text
- `#64748B` secondary text
- `#D63D36` Humungousaur identity red
- `#246BFE` planning blue
- `#10A37F` memory/completion green
- `#F1F5F9` soft surface

## Typography

- Prefer Inter, SF Pro, or system UI sans-serif.
- Use JetBrains Mono or SF Mono only for compact tool labels.

## Motion Rules

- Use smooth, purposeful step transitions.
- Keep all labels readable at every sampled frame.
- Emphasize the active cognition stage with scale, color, and connector glow.
- No chaotic particle fields, no dark cyberpunk palette, no decorative blobs, and no unreadable microtext.
