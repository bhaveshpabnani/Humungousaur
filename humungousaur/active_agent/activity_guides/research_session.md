# Activity Skill Pack: Research Session

## Summary

Use when the user is gathering, comparing, reading, saving, or organizing source
material. Optimize for source trails, task continuity, and careful approval
before content inspection.

## Signals

- Sustained navigation across related browser pages, knowledge bases, files, or
  saved references.
- Searches, bookmarks, downloads, reading-list saves, notes, highlights, or
  copied references clustered around a topic.
- Switching between sources and a work artifact such as a document, issue,
  spreadsheet, deck, or message.

## Helpful Moments

- The user returns to a research cluster after a gap.
- Many sources accumulate and a safe source trail or comparison frame may help.
- Research appears to feed a document, decision, purchase, issue, or follow-up.
- The user explicitly asks for synthesis, source quality, or next steps.

## Stay Silent When

- Pages are briefly opened, unrelated, private, policy-blocked, or incidental.
- Assistance would require exact URLs, titles, queries, page text, account data,
  or private search context without approval.
- The user is browsing normally with no safe interruption boundary.

## Deep Dive Triggers

- Reading page text, files, query text, full URLs, titles, saved notes, or
  downloaded content.
- Summarizing, comparing, ranking, extracting citations, or checking claims from
  source content.
- Opening additional pages or performing searches on the user's behalf.

## Memory Guidance

- Store redacted source hashes, broad source types, source count, related artifact
  hashes, topic labels from safe metadata, and whether research led to notes,
  downloads, decisions, or work artifacts.
- Remember user-declared evaluation criteria and explicit decisions.
- Do not retain raw page text, queries, URLs, titles, quotes, or personal account
  context without approval.

## Privacy Notes

- Prefer source clustering over content reading.
- Ask before inspecting source content or using exact browsing details.
- Treat research topics as potentially sensitive.
