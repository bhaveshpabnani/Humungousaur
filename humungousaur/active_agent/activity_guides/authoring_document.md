# Activity Skill Pack: Document Authoring

## Summary

Use when the user is creating, editing, reviewing, exporting, or publishing a
document-like artifact. Optimize for continuity, source awareness, and quiet
resume help.

## Signals

- Sustained editing, comments, suggestions, outline changes, saves, exports, or
  shares on a document surface.
- Repeated switching between a document and source material, notes, messages, or
  file assets.
- Copy, paste, citation, attachment, or review activity clustered around the same
  document entity.

## Helpful Moments

- The user returns after a gap and may need a concise resume capsule.
- A draft appears ready for review, export, sharing, or follow-up.
- The same source or section is revisited repeatedly and a blocker may exist.
- The user explicitly asks for structure, wording, citations, or next steps.

## Stay Silent When

- The document is only previewed briefly or updated by background sync.
- The surface is private, credential, payment, legal-sensitive, health-sensitive,
  or otherwise policy-restricted.
- Assistance would require raw document text, comments, names, or links that were
  not approved for inspection.
- The user is in uninterrupted writing flow with no safe boundary.

## Deep Dive Triggers

- Reading document body, comments, revision history, attachments, or source pages.
- Comparing source material against claims or citations.
- Preparing edits, summaries, review notes, or publication-ready text from
  non-metadata content.

## Memory Guidance

- Store redacted document/entity hashes, broad artifact type, safe source counts,
  last safe action, user-declared goal, and completion signals.
- Remember blockers, review state, and export/share outcomes only from safe
  metadata or explicit user input.
- Do not retain raw prose, comments, names, URLs, or copied snippets without
  approval.

## Privacy Notes

- Prefer metadata-level continuity over content inspection.
- Ask before reading or generating from document content.
- Treat shared, confidential, personal, and client documents as sensitive by
  default.
